Proyecto Ahorcado

Nombre del proyecto:

Ahorcado - Edición Mundial 2026 ⚽🏆

Integrantes:
Jafet Lombeida

Objetivo del sistema:
El objetivo de este proyecto es poner en práctica los conocimientos adquiridos en las primeras semanas de aprendizaje del lenguaje Python, dentro del proceso de formación en la carrera de Ciberseguridad.

A través del desarrollo de este pequeño juego se busca:
- Reforzar el uso de estructuras de control (bucles while, condicionales if/elif/else).
- Comprender el manejo de listas y su uso para almacenar información durante la ejecución del programa.
- Practicar la interacción con el usuario mediante la entrada y salida de datos.
- Familiarizarme con la lógica de programación y el pensamiento algorítmico, como parte de los primeros pasos en este mundo de la programación.


Descripción de funcionalidades

El programa simula el clásico juego del Ahorcado, adaptado con la temática del Mundial 2026, donde el usuario debe adivinar el nombre de un país participante.
Sus principales funcionalidades son:

Selección aleatoria de palabra: el programa elige al azar, de una lista predefinida de selecciones (Ecuador, Colombia, Alemania, Argentina, Brasil, Francia, España, Portugal), el país que el usuario deberá adivinar.
Sistema de intentos: el jugador cuenta con 6 intentos para adivinar la palabra correctamente.
Visualización del progreso: en cada ronda se muestra la palabra oculta, revelando únicamente las letras que el usuario ha acertado hasta el momento.
Registro de letras incorrectas: el sistema lleva un control de las letras que ya fueron ingresadas y que no forman parte de la palabra, mostrándolas en pantalla.
Validación de letras repetidas: si el usuario ingresa una letra ya utilizada (correcta o incorrecta), el programa lo notifica y no resta intentos.
Condición de victoria o derrota: el juego finaliza cuando el usuario adivina todas las letras de la palabra (victoria) o cuando se agotan los 6 intentos (derrota), mostrando en ambos casos un mensaje final junto con el nombre del equipo que debía adivinarse.
